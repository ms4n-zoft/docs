# Zoftware Stakeholder Updates

Mintlify documentation portal for stakeholder visibility across Zoftware product initiatives.

## Local preview

```bash
mint dev
```

## Structure

- `docs.json` controls branding and navigation.
- `index.mdx` is the main Whitelabeling Solution dashboard.
- `whitelabeling/*.mdx` contains one page per workstream.
- `release-notes.mdx` contains stakeholder-facing update history.
